How Mastery Level Is Decided

What is mastery level?

Each grammar point has a mastery level. The app uses your correct and wrong answers to decide whether a grammar point is new, learning, weak, or mastered.

Mastery is decided per grammar point, not per question. This means if one grammar point appears in several questions, all answers for that grammar point affect the same mastery level.

The counts used by the app

The app keeps two main counts for each grammar point:

Count Meaning
Correct count How many times you answered this grammar point correctly.
Wrong count How many times you answered this grammar point wrongly.
Accuracy Correct count divided by total answers for that grammar point.

Mastery rules

After each answer, the app recalculates the mastery level using these rules.

Mastery level Rule Simple meaning
New No correct answer yet, and wrong answers are fewer than 3. You have not practiced this grammar point enough yet.
Learning At least 1 correct answer, but not mastered and not weak. You have started learning this grammar point.
Weak Wrong count is 3 or more, and accuracy is below 80%. You made repeated mistakes, so this grammar point needs review.
Mastered Correct count is 5 or more, and accuracy is 90% or higher. You answered this grammar point correctly many times with high accuracy.

Examples

Correct Wrong Accuracy Mastery Why?
0 1 0% New Wrong answers are still fewer than 3.
0 3 0% Weak Wrong count is 3 or more and accuracy is below 80%.
2 1 66.7% Learning You have correct answers, but not enough to be mastered.
5 0 100% Mastered Correct count is 5 or more and accuracy is 90% or higher.
8 3 72.7% Weak Wrong count is 3 or more and accuracy is below 80%.

Important note

A grammar point can move between mastery levels. For example, a weak grammar point can become learning or mastered later if your accuracy improves.

The app updates the mastery level after every submitted answer.
